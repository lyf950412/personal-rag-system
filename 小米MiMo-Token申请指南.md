## 2️⃣ 项目描述（重点填写，越详细越好）

```
本项目是一个基于Spring Boot + React的企业级RAG（检索增强生成）知识库系统，旨在为企业提供智能化的文档管理和知识检索解决方案。

核心功能：
1. 多格式文档处理：支持PDF、Word、Excel、PPT、TXT等多种格式的自动解析和分块处理
2. 智能向量化存储：使用PostgreSQL + pgvector进行高效的向量存储和检索
3. RAG智能问答：基于检索增强生成技术，实现精准的知识问答和答案追溯
4. 用户认证系统：JWT Token认证，支持用户注册/登录/权限管理
5. 知识库管理：支持多知识库创建、文档管理、统计分析

技术架构：
- 后端：Spring Boot 3.2.0 + Spring AI 1.0.0-M6 + Spring Security
- 前端：React 18 + Vite 5 + Ant Design 5
- 数据库：PostgreSQL + pgvector向量数据库
- AI集成：支持火山引擎Doubao系列Embedding模型
- 文档解析：Apache Tika + Spring AI DocumentReader

应用场景：
- 企业内部知识库管理
- 智能客服系统
- 文档检索与问答
- 教育培训平台
- 技术文档管理
```

### Q1: 没收到邮件怎么办？

- 确认填写的邮箱地址是否正确

- 如仍未收到，请通过开放平台「联系我们」进行反馈

### Q2: 权益有有效期吗？
- 有。不同权益的有效期以到账提示为准，过期未使用的权益将自动失效，请及时使用和体验

### Q3: 我会获得什么权益？
- 根据表单填写内容和开放平台账号状态，发放不同额度、不同档位的Token Plan或赠金
- Token Plan是开放平台的订阅制套餐，可直接配置到Claude Code、Cursor等编程工具中使用
- 赠金将直接充入开放平台账户余额，用于按量计费的API调用消费

### Q4: 确保权益顺利发放的注意事项
- 已有开放平台账号？请确保它已由你在表单中填写的邮箱注册
- 如果是由手机号注册的，请绑定该邮箱
- 还没有开放平台账号？请在收到评估通过的邮件后，根据邮件提示使用填写的邮箱完成注册
